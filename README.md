# King Burger

Projeto front-end do King Burger — layout estático com HTML, CSS e JavaScript.

## Descrição

Página de apresentação e catálogo do King Burger com seções de menu, hero, rodapé e detalhes de produto.

## Estrutura do projeto

- index.html — página principal
- css/ — arquivos de estilo (ex.: `styles.css`, `menu.css`)
- js/ — scripts de interação (ex.: `main.js`, `menu.js`)
- images/ — ativos de imagens

## Como abrir

1. Abrir `index.html` no navegador (duplo clique) ou usar uma extensão "Live Server".

## Desenvolvimento

- Faça mudanças nos arquivos em `css/` e `js/`.
- Recarregue a página no navegador para ver as alterações.

## Tecnologias

- HTML5, CSS3, JavaScript (ES6)

## Observações

- Este repositório contém apenas frontend estático; não há build ou dependências.
